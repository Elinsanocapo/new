Reglas del Juego:

1. Haz clic en una nave para seleccionarla.

2. Usa las flechas del teclado para mover la nave seleccionada.

3. Haz clic en el cubo para girarlo y mostrar un número aleatorio en cada tirada.

4. El juego termina cuando las naves alcanzan sus destinos.
